# Análisis de Vehículos en USA

Esta aplicación web permite revisar datos de vehículos usados en Estados Unidos.

## Funcionalidad

- Visualización de la **distribución de vehículos por año de modelo**  permitiendo identificar tendencias y patrones.
- Interactividad mediante checkboxes, para mostrar los gráficos que el usuario desee.
- Tecnología utilizada: Python, Streamlit, Plotly Express, Pandas.
